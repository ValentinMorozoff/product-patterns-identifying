# Выявление закономерностей успешного продукта на примере компьютерных игр

## О проекте

Это исследовательский проект по product analytics на исторических данных о продажах видеоигр, пользовательских и экспертных оценках, жанрах, платформах и рейтингах.

Цель проекта — выделить факторы, которые связаны с коммерческой успешностью игрового продукта.

## Что внутри репозитория

- `successful_product_patterns.ipynb` — основной ноутбук исследования;
- `data/games.csv` — исходный датасет;
- `README.md` — описание проекта;
- `CONTRIBUTING.md` — правила поддержания репозитория.

## Стек

- Python
- Jupyter Notebook
- pandas
- NumPy
- SciPy
- matplotlib
- seaborn

## Данные

Исходные данные находятся в репозитории:

- `data/games.csv`

Ноутбук сначала ищет локальный файл в папке `data/`, а затем может использовать legacy-путь старой учебной среды.

## Ключевой результат

В проекте анализируются:

- динамика рынка по годам;
- различия между платформами и регионами;
- влияние жанра, рейтингов и оценок на продажи;
- продуктовые паттерны, связанные с устойчивым коммерческим результатом.

## Как открыть проект

1. Клонируй репозиторий.
2. Запусти Jupyter Notebook или JupyterLab.
3. Открой `successful_product_patterns.ipynb`.
4. Убедись, что файл `data/games.csv` доступен локально.

## Формат проекта

Репозиторий оформлен как portfolio-case:

- код — на английском языке;
- описание логики исследования и выводы — на русском;
- структура ориентирована на читабельность и повторяемость анализа.
